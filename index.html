개발 배경

발표가 두렵지 않도록,
소리를 눈으로 보다

청각장애 대학생은 자신의 발음이 타인에게 어떻게 들리는지 확인할 방법이 없습니다. 발표를 앞두고 느끼는 불안감은 비장애인과는 차원이 다릅니다.

말결은 이 문제를 해결하기 위해 탄생했습니다. 시각적 피드백 하나로 발표에 대한 자신감을 되찾을 수 있도록, 산업디자인 전공의 감성과 기술을 결합했습니다.

01
문제 발견
청각장애 대학생이 발표 전 자신의 발음을 직접 확인할 수 있는 도구가 없다는 문제를 발견했습니다.
02
아이디어
음파 시각화와 STT 비교를 결합하면 청각 정보를 시각 정보로 변환할 수 있다는 아이디어에서 출발했습니다.
03
유니버설 디자인
장애인만을 위한 서비스가 아닌, 발표 불안을 가진 모든 대학생이 사용할 수 있는 배리어프리 서비스를 목표로 합니다.

Q1
나의 아이디어를 한 줄로 소개해주세요.
필수
청각장애 대학생이 발표 전 자신의 발음과 전달력을 시각적으로 확인하고 연습할 수 있는 발표 리허설 서비스입니다.
Q2
아이디어를 떠올린 배경 이야기를 들려주세요.
필수

대학교 근로 도우미로 활동하면서 같은 학부의 청각장애 학생과 수업을 함께 듣게 되었습니다. 발표 상황에서 발음이 새거나 전달이 잘 되지 않아 어려움을 겪는 모습을 보았고, 청각장애 학생이 발표 전에 자신의 발음이 상대방에게 어떻게 들리는지 스스로 확인하기 어렵다고 느꼈습니다.

기존의 발음 교정 앱은 청각장애 대학생의 실제 대학생활 맥락에 맞춘 발표 연습 공간이 부족합니다. 그래서 발음을 완벽하게 교정하는 서비스가 아니라, 발표 전에 자신의 말이 얼마나 전달되는지 시각적으로 점검하고 반복 연습할 수 있는 서비스를 기획하게 되었습니다.

Q3
아이디어는 누구의 어떤 문제를 해결해주나요?
필수

이 아이디어는 발표와 구두 설명이 필요한 청각장애 대학생의 문제를 해결합니다.

청각장애 대학생은 자신의 발음이나 말의 전달력을 스스로 객관적으로 확인하기 어렵고, 발표 전 피드백을 받을 사람도 항상 곁에 있지 않습니다.

이 서비스는 발표 대본을 입력하고 녹음하면, 원래 대본과 음성 인식 결과를 비교해 잘 전달된 문장과 다시 연습이 필요한 단어를 시각적으로 보여줍니다.

Q4
아이디어를 어떻게 실현하고 싶으신가요?
필수

MVP 단계에서는 발표 대본 입력 → 발표 녹음 → 음성 인식 결과 비교 → 어려운 단어 표시 → 반복 연습 리포트 제공 구조로 실현합니다.

주요 기능: 발표 대본 입력 / 문장별 녹음 / 원문과 음성 인식 결과 비교 / 전달이 어려운 단어 표시 / 발표 전 최종 리포트 제공
