배리어프리 발표 리허설 도우미

내 목소리를 눈으로 확인하세요

말결은 청각장애 대학생이 발표 전에 자신의 목소리 전달력과 발음을 직관적으로 확인하고 반복 훈련할 수 있는 솔루션입니다. 소리를 파형 음파와 텍스트 일치율로 시각화하여 배리어프리 연습 기회를 선사합니다.

이렇게 사용하세요

01
대본 입력
발표하고 싶은 스크립트를 작성하거나 붙여넣습니다.
02
녹화 시작
녹음 단추를 누르고 소리내어 대본을 끝까지 낭독해보세요.
03
비교 확인
음성 텍스트 변환 결과와 원래 대본의 일치율을 단어 단위로 비교 분석합니다.
04
반복 연습
미진하게 발음된 부분을 눈으로 확인하고 피드백 리포트에 따라 반복 연습합니다.

말결의 핵심 기능

〰️
실시간 음파 시각화
Web Audio API를 사용해 실제 마이크로 입력되는 소리의 크기와 박자를 부드러운 음파 형태로 생생하게 보여줍니다.
📝
STT 대본 비교 분석
웹 음성 인식 API(Web Speech API)를 활용해 소리를 정확히 식별한 뒤 원래 스크립트와 대조하여 전달 상태를 색깔로 구분합니다.
📊
자동 종합 점수 계산
발음의 일치율, 적당한 말 속도(분당 글자수, CPM), 흐름의 끊김 없는 연속성을 종합하여 100점 만점으로 정밀 평가합니다.
🗂
연습 기록 브라우저 저장
서버에 개인 데이터가 흘러가지 않아 안심할 수 있으며, 기기의 브라우저 내부(localStorage)에 자동으로 축적되어 기록을 누적 비교할 수 있습니다.
